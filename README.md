# SlideR
SlideR - Lite Presentation Management Tool for Android Devices
SlideR, bilgisayara bağlı kalmadan slaytlar arasında geçiş yapmanızı sağlayan bir uygulamadır.
İki parçadan oluşmaktadır.
1- Host Uygulaması
Sunumun gösterildiği makineye kurulan host uygulaması, android uygulamasıyla haberleşmek üzere sunucu makine üzerinde bir kanal açar.
2- Mobil Uygulama
Host ile iletişim kurarak, sunucu makine üzerinde slayt geçişi komutlarını çalıştırmanıza olanak sağlar.
PROGRAMIN KISITLARI
1- Host uygulaması, şu an için yalnızca Windows işletim sistemine sahip cihazlarda çalışmaktadır.
2- Mobil uygulama, şu an için yalnızca Android işletim sistemine sahip cihazlarda çalışmaktadır.
3- Sunucu makinenin ve android cihazın aynı yerel ağa bağlı olması gerekmektedir.
4- Mobil uygulama ile slayt geçişlerini yapabilmeniz için, sunucu makine üzerinde sunum yaptığınız uygulamanın seçili olması (focused) gerekmektedir.
ÖNEMLİ!!!
Klavye üzerindeki Sağ-sol yön tuşları ile slaytlarda geçiş yapabildiğiniz tüm sunum programlarında bu uygulamayı kullanabilirsiniz.
